Skills

This bootcamp is divided into two phases. Please see below what I expect to learn from this bootcamp.

Phase 1: Front-End Foundations

The Front-End Foundations phase covers the fundamental concepts of web development, including front-end technologies, command-line fundamentals, user experience design, and APIs. In the Front-End Foundations phase, I will learn the following:

  • Command-line fundamentals
  • Git and GitHub
  • HTML and CSS
  • User experience design
  • JavaScript
  • Third-party APIs (such as jQuery and Bootstrap)
  • Server-side APIs, AJAX, and JSON
  • Agile software development methodology

Phase 2: Front-End Frameworks

In the Front-End Frameworks phase, you'll learn the skills necessary to develop a portfolio of web applications showcasing your front-end skills, using front-end libraries and testing frameworks. You'll also learn to deploy these applications to interact with real-world users. In the Front-End Frameworks phase, I will learn the following:

  • Node.js
  • Test-driven development
  • Object-oriented programming (OOP)
  • React
  • Behavior-driven development
  • Deployment
  • How to create a professional project portfolio
